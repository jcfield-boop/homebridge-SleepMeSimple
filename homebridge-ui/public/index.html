Warm Hug Feature

The Warm Hug feature gradually increases temperature over time, starting before your scheduled time, for a gentle wake-up experience. When selected, your device will begin warming before the time you set to ensure it reaches your target temperature exactly at your scheduled time.

For detailed configuration of Warm Hug parameters, please see the "Warm Hug Options" tab.

Advanced Template Editing

Templates create predefined schedule patterns based on sleep science. There are three ways to customize templates:

Option 1: Edit Individual Schedules

▾

After applying templates, you can edit each schedule individually using the "Edit" button. Template-based schedules are marked with a "T" badge.

1. Apply a template 2. Click "Edit" on any schedule 3. Adjust values as needed 4. Click "Update Schedule"

Option 3: Create Custom Templates

▾

Advanced users can create custom templates by editing the config.json file directly:

  1. Stop Homebridge
  2. Open the config.json file (typically in ~/.homebridge/config.json)
  3. Find the SleepMeSimple platform configuration
  4. Add or modify the "templates" property
  5. Save the file and restart Homebridge

Example of a custom template in config.json:

{
    "platform": "SleepMeSimple",
    "name": "SleepMe Simple",
    "apiToken": "your-token-here",
    "unit": "C",
    "pollingInterval": 90,
    "enableSchedules": true,
    "schedules": [...],
    "templates": {
      "myCustom": {
        "name": "My Custom Template", 
        "description": "Custom temperature profile",
        "schedules": [
          { "type": "Weekdays", "time": "22:00", "temperature": 21, "description": "Evening Cool Down" },
          { "type": "Weekdays", "time": "23:30", "temperature": 19, "description": "Deep Sleep" },
          { "type": "Weekdays", "time": "03:00", "temperature": 22, "description": "REM Support" },
          { "type": "Weekdays", "time": "06:30", "temperature": 25, "description": "Warm Hug Wake-up" }
        ]
      }
    }
  }

▾

How Warm Hug Works

▾

When you create a "Warm Hug" schedule type, the device will:

  1. Start at your current temperature
  2. Begin warming up gradually before your scheduled time (based on the duration setting)
  3. Increase temperature at the specified rate until reaching your target temperature
  4. Reach the exact target temperature at precisely your scheduled time

This creates a gentle, natural wake-up experience similar to sunrise, helping you start your day refreshed.
